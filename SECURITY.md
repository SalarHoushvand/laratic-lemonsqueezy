# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please email **laratic@mail.com** with:

- A description of the issue
- Steps to reproduce
- Any potential impact

Please do **not** open public GitHub issues for security vulnerabilities.

## Response Timeline

- **Acknowledgment:** within 48 hours
- **Initial assessment:** within 7 days
- **Fix or mitigation plan:** communicated as soon as a path forward is identified

We appreciate responsible disclosure and will credit reporters when fixes are published (unless you prefer to remain anonymous).
